git_submodule_update_strategy

Version:

Get the update rule that will be used for the submodule.

This value controls the behavior of the git submodule update command. There are four useful values documented with git_submodule_update_t.

Signature

GitHub
git_submodule_update_t git_submodule_update_strategy(git_submodule *submodule);

Parameters

git_submodule *
In
submodule

The submodule to check

Returns

git_submodule_update_t

The current git_submodule_update_t value that will be used for this submodule.
