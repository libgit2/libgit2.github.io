git_filter_list_load_ext

Version:

Load the filter list for a given path.

This will return 0 (success) but set the output git_filter_list to NULL if no filters are requested for the given file.

Signature

GitHub
int git_filter_list_load_ext(git_filter_list **filters, git_repository *repo, git_blob *blob, const char *path, git_filter_mode_t mode, git_filter_options *opts);

Parameters

git_blob *

const char *
In

git_filter_mode_t

git_filter_options *

Returns
