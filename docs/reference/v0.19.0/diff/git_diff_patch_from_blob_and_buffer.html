git_diff_patch_from_blob_and_buffer

Version:

Directly generate a patch from the difference between a blob and a buffer.

This is just like git_diff_blob_to_buffer() except it generates a patch object for the difference instead of directly making callbacks. You can use the standard git_diff_patch accessor functions to read the patch data, and you must call git_diff_patch_free() on the patch when done.
