patch

Version:
Patch handling routines.

Objects

git_patch

The diff patch is used to store all the text diffs for a delta

Functions

git_patch_owner

Get the repository associated with this patch

git_patch_from_diff

Return a patch for an entry in the diff list

git_patch_from_blobs

Directly generate a patch from the difference between two blobs

git_patch_from_blob_and_buffer

Directly generate a patch from the difference between a blob and a buffer

git_patch_from_buffers

Directly generate a patch from the difference between two buffers

git_patch_free

Free a git_patch object

git_patch_get_delta

Get the delta associated with a patch data and you do not have to release it when you are done with it.

git_patch_num_hunks

Get the number of hunks in a patch

git_patch_line_stats

Get line counts of each type in a patch

git_patch_get_hunk

Get the information about a hunk in a patch

git_patch_num_lines_in_hunk

Get the number of lines in a hunk

git_patch_get_line_in_hunk

Get data about a line in a hunk of a patch

git_patch_size

Look up size of patch diff data in bytes

git_patch_print

Serialize the patch to text via callback

git_patch_to_buf

Get the content of a patch as a single diff text
