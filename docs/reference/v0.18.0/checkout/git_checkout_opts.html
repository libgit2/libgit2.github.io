git_checkout_opts

Version:

Checkout options structure

Zero out for defaults. Initialize with GIT_CHECKOUT_OPTS_INIT macro to correctly set the version field. E.g.

git_checkout_opts opts = GIT_CHECKOUT_OPTS_INIT;
