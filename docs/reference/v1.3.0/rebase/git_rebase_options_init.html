git_rebase_options_init

Version:

Initialize git_rebase_options structure

Initializes a git_rebase_options with default values. Equivalent to creating an instance with GIT_REBASE_OPTIONS_INIT.

Signature

GitHub
int git_rebase_options_init(git_rebase_options *opts, unsigned int version);

Parameters

git_rebase_options *
In
opts

The git_rebase_options struct to initialize.

unsigned int
In
version

The struct version; pass GIT_REBASE_OPTIONS_VERSION.

Returns

int

Zero on success; -1 on failure.
