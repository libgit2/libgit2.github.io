# [libgit2's web site](http://libgit2.org/)

The reference documentation is automatically generated from the in-line documentation in libgit2's public header files. This generated content is in the [www.libgit2.org-docs](https://github.com/libgit2/www.libgit2.org-docs) repository, and included as a submodule in the `generated-docs` folder. To ensure that you have it, `git clone --recursive` this repository.

```bash
bundle install
bundle exec jekyll serve --watch
```
